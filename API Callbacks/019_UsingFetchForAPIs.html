Fetch is a new way for making HTTP request.
It supports promises and hence prevent callback hells
fetch( "url" ); = fetch is an inbuit function which expects a url as an argument and return a promise.
but..........
whenever fetch is executed with a url it returns response as soon as headers are fetched
and we do not get the responseText with it.

To get back the responseText of the data we again apply a method to the response we got earlier as
response.json() this method also returns a promise.

A third way of HTTP request is by using a third party library called AXIOs
AXIOs is a library for making HTTP request
AXIOs uses fetch in their backend for making HTTP request
